Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]
```

Слайс b создается как срез массива a с индексами от 1 до 4 не включительно, и содержит элементы 77, 78, 79 из исходного массива a
